# passport-with-mongodb

A website demostrates passportJS authentication (local strategy) with mongoDB (noSQL database).

### NPM Packages

##### Dependencies:
- bcryptjs
- connect-flash
- express-session
- mongodb
- mongoose
- passport
- passport-local
##### Dev Dependencies [--save-dev]:
- nodemon 

### Preview this project

```sh
$ cd passport-with-mongodb
$ npm i
$ nodemon app.js
$ go to localhost:8080
```
